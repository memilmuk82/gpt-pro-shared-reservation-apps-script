# SECURITY_AND_PRIVACY.md

## 핵심 보안 원칙

- GPT 계정 ID 저장 금지
- GPT 비밀번호 저장 금지
- 예약 시스템에 공용 계정 비밀번호 입력 금지
- GPT 접속 정보는 학교 업무게시판에서 관리
- 예약 시스템은 업무게시판 참고 안내만 표시
- 개인정보/민감정보/평가자료/학생부 자료 입력 금지 안내 유지

## 사용자 인증/권한

- Google Workspace 접속 계정 기준
- `senedu.kr` 도메인 기준
- 사용자는 Users 시트에 등록되어야 시스템 사용 가능
- 비활성 사용자는 예약 시스템 사용 불가
- 관리자 함수는 서버에서 권한 검증 수행
- 최소 1명의 활성 admin 유지

## CSV 일괄 등록 보안

- 프론트 파싱은 UX 보조일 뿐 최종 검증이 아님
- 서버에서 도메인, 중복, 권한, 활성값, 정렬값 검증
- 하나라도 실패하면 아무 사용자도 저장하지 않음
- 실제 사용자 CSV는 GitHub에 올리지 않음

## XSS 방지

- 사용자 입력값은 화면 출력 시 escape 처리
- GuideItems HTML 문자열은 실행하지 않고 텍스트로 표시
- Settings 안내 문구도 화면 출력 시 escape 처리
- innerHTML 사용 시 신뢰할 수 없는 값은 직접 삽입하지 않음

## 공용 PC 주의

앱의 화면 잠금은 Google 계정 또는 GPT 계정 자체 로그아웃이 아니다.

사용자는 사용 종료 후 다음을 직접 확인해야 한다.

```text
1. GPT 로그아웃
2. Google 계정 로그아웃
3. 브라우저에 이전 화면이 남아 있지 않은지 확인
4. 예약 시스템 화면 잠금 확인
```
